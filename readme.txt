To use the macro:

1) Paste contents of SummaryStatements.vba into word macro and save.
2) A blank word document should be created in a folder.  The folder should also contain a child directory called ToProcess.
3) Summary statements generated in P2RMIS should be placed in the ToProcess folder.
4) Run the macro ProcessSummaryStatementDirectory. Formatted word documents will be located in a /doc subdirectory, pdfs in /pdf subdirectory.
